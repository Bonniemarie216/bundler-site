Contributing to Bundler

Welcome to Bundler! We are so happy that you’re here. We know joining a new open source project can be daunting, so here’s a quick overview of what you can expect from this documentation.

Something missing? Send us a pull request!

Recommended first steps

  • Join us in the Bundler slack! Generate an invite here
  • If you’re interested in helping with code:
    • Get a quick overview of our development process
    • Setup your machine for development
    • Checkout How you can help: your first contributions! for a list of suggestions to get started
  • If you’re interested in helping with documentation:
    • Read up on how documentation works for Bundler
    • Learn about our documentation vision
