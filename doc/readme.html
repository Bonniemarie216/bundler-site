Docs: Contributing and developing Bundler

If you’re looking for documentation on how to use Bundler: visit bundler.io, or run bundle help from the command line. You may also be interested in troubleshooting common issues found when using Bundler.

Bundler welcomes contributions from everyone. While contributing, please follow the project code of conduct, so that everyone can be included.

If you’d like to help make Bundler better, you totally rock! Thanks for helping us make Bundler better.
