Bundler 1.13 and Redesigned bundler.io

by xxxx on Jun 27 2016

Bundler v1.13

Redesigned bundler.io

Most visible change is completely redesigned page with new styles and graphics. Now it’s responsive and adjusted also for mobile devices using Bootstrap framework.

The Docs page allows you to choose between guide, command reference or versions’ changelog. After choosing one of commands (e.g. bundle install), there is a new sidebar, making navigation between other commands and versions easier.

There are also two new guides: Using Bundler In Application written by Jakub (@kruczjak) and Developing a RubyGem using Bundler by Ryan (@radar).

More improvements:

  • Man pages from Bundler repository instead of hand-written commands pages (where possible)
  • Commit to master branch in bundler-site repository causes auto-deploy of the newest page version (via Travis)
  • Middleman updated to latest version
  • Added anchor links in guides and commands docs for faster navigation
  • [WIP] Started work on translations

Many thanks to Amy (@sailorhg), Andre (@indirect) and Samuel (@segiddins). Without their help, it wouldn’t be possible.

Enjoy :D

~ Jakub (@kruczjak)
